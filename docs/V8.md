# V8 Bot Client And Project Closure

## Goal

V8 adds a reproducible client-side load entry for the V1 server.

The target is not large-scale benchmarking. The target is to prove the current server loop can be exercised by multiple TCP clients.

## Included

- `rrs_bot_client` executable
- 16 concurrent bot clients
- 4 room distribution
- JOIN / INPUT / RECONNECT command flow
- Snapshot receive counting
- Bot-side summary report

## Not Included

- Web dashboard
- Large-scale load testing
- Real packet protocol
- Binary serialization
- Graph generation

## Usage

Start the server:

```bash
./build/realtime_room_server
```

Run bots in another terminal:

```bash
./build/rrs_bot_client
```

Expected bot report:

```text
[BotReport] bots=16 connected=16 joined=16 inputsSent=... snapshotsReceived=... reconnects=4 errors=0
```
