# V7 Async Persistence And Metrics

## Goal

V7 adds non-realtime side paths without blocking the room tick loop.

```text
IO / Worker event -> PersistenceWorker queue -> local event file
Worker / IO stats -> MetricsReporter -> periodic log
```

## Included

- `PersistenceEvent`
- `PersistenceWorker`
- File-backed async event sink
- `MetricsReporter`
- IO events: client connected, client disconnected, joined, reconnected
- Worker events: bind player, stale input rejected
- Periodic metrics summary log

## Not Included

- MySQL
- Kafka
- Redis
- Prometheus exporter
- Exactly-once persistence
- Event replay

## Manual Test

Start the server and connect a client:

```bash
./build/realtime_room_server
```

```bash
nc 127.0.0.1 9000
JOIN 1 1001
INPUT 1 0
RECONNECT
QUIT
```

Verify:

- `logs/persistence_events.log` contains event lines
- `[Metrics]` lines are printed periodically
- Room tick still runs at 30Hz
