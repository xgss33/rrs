# V0 Project Skeleton

## Goal

V0 only verifies that the project can compile, start, initialize basic infrastructure, and print startup logs.

## Included

- C++20 build setup
- Basic directory layout
- Strong domain id wrappers
- App config defaults
- Logger facade
- Minimal `main`

## Not Included

- TCP network
- Message queue
- Snapshot generation
- Persistence

Worker thread loop and room tick are introduced in V1.
